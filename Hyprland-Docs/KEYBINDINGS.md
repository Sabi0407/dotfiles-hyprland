# Guide des raccourcis clavier Hyprland

Configuration des raccourcis clavier pour Hyprland.  
Touche principale : `Super` (touche Windows)

---

## Applications et outils

### Applications principales

| Raccourci | Action |
|-----------|--------|
| `Super` + `Enter` | Ouvrir le terminal |
| `Super` + `E` | Gestionnaire de fichiers |
| `Super` + `Tab` | Lanceur d'applications |

### Navigateurs web

| Raccourci | Action |
|-----------|--------|
| `Super` + `B` | Firefox |
| `Super` + `Shift` + `B` | Firefox en mode privé |
| `Super` + `Ctrl` + `B` | Brave en mode incognito |

### Outils multimédia et productivité

| Raccourci | Action |
|-----------|--------|
| `Super` + `O` | Obsidian |
| `Super` + `M` | Sélecteur de fichiers MPV |
| `Super` + `K` | Kooha (capture vidéo) |
| `Super` + `Z` | Zathura (lecteur PDF) |

### Presse-papiers

| Raccourci | Action |
|-----------|--------|
| `Super` + `Ctrl` + `V` | Gestionnaire de presse-papiers |
| `Super` + `Alt` + `Delete` | Supprimer l'historique du presse-papiers |

---

## Gestion des fenêtres

### Commandes générales

| Raccourci | Action |
|-----------|--------|
| `Super` + `X` | Fermer la fenêtre active |
| `Super` + `V` | Basculer entre mode flottant et tiling |
| `Super` + `F` | Basculer en plein écran |
| `Super` + `D` | Masquer/Afficher toutes les fenêtres |
| `Super` + `C` | Centrer toutes les fenêtres flottantes |

### Groupes de fenêtres (onglets)

| Raccourci | Action |
|-----------|--------|
| `Super` + `G` | Créer ou dissoudre un groupe |
| `Super` + `Shift` + `G` | Verrouiller/Déverrouiller le groupe |
| `Super` + `Ctrl` + `G` | Sortir la fenêtre du groupe |
| `Super` + `Alt` + `G` | Ajouter la fenêtre au groupe à droite |
| `Alt` + `H` | Onglet précédent |
| `Alt` + `L` | Onglet suivant |
| `Super` + `'` | Alterner entre les onglets |

### Layouts et disposition

| Raccourci | Action |
|-----------|--------|
| `Super` + `P` | Activer le pseudo-tiling |
| `Super` + `J` | Basculer entre split vertical et horizontal |
| `Super` + `Alt` + `Enter` | Égaliser la disposition des fenêtres |
| `Super` + `Y` | Basculer entre layout Dwindle et Master |

### Navigation

| Raccourci | Action |
|-----------|--------|
| `Super` + `←` | Focus sur la fenêtre à gauche |
| `Super` + `→` | Focus sur la fenêtre à droite |
| `Super` + `↑` | Focus sur la fenêtre en haut |
| `Super` + `↓` | Focus sur la fenêtre en bas |

### Déplacement de fenêtres

| Raccourci | Action |
|-----------|--------|
| `Super` + `Shift` + `←` | Déplacer la fenêtre à gauche (dans groupe) |
| `Super` + `Shift` + `→` | Déplacer la fenêtre à droite (dans groupe) |
| `Super` + `Shift` + `↑` | Déplacer la fenêtre en haut (dans groupe) |
| `Super` + `Shift` + `↓` | Déplacer la fenêtre en bas (dans groupe) |
| `Super` + `Alt` + `←` | Déplacer la fenêtre à gauche |
| `Super` + `Alt` + `→` | Déplacer la fenêtre à droite |
| `Super` + `Alt` + `↑` | Déplacer la fenêtre en haut |
| `Super` + `Alt` + `↓` | Déplacer la fenêtre en bas |

### Redimensionnement

| Raccourci | Action |
|-----------|--------|
| `Super` + `Ctrl` + `←` | Réduire la largeur de 50px |
| `Super` + `Ctrl` + `→` | Augmenter la largeur de 50px |
| `Super` + `Ctrl` + `↑` | Réduire la hauteur de 50px |
| `Super` + `Ctrl` + `↓` | Augmenter la hauteur de 50px |
| `Super` + `Molette haut` | Réduire la hauteur de 10px |
| `Super` + `Molette bas` | Augmenter la hauteur de 10px |
| `Alt` + `Molette haut` | Réduire la largeur de 10px |
| `Alt` + `Molette bas` | Augmenter la largeur de 10px |

### Contrôles à la souris

| Raccourci | Action |
|-----------|--------|
| `Super` + `Clic gauche` | Déplacer la fenêtre |
| `Super` + `Clic droit` | Redimensionner la fenêtre |

## Espaces de travail

### Scratchpad

| Raccourci | Action |
|-----------|--------|
| `Super` + `I` | Afficher/Masquer le scratchpad |
| `Super` + `Shift` + `I` | Déplacer la fenêtre vers le scratchpad |

### Navigation entre workspaces

| Raccourci | Action |
|-----------|--------|
| `Super` + `1` à `0` | Aller au workspace 1 à 10 |
| `Super` + `Shift` + `Molette` | Défiler entre les workspaces |

### Déplacement de fenêtres vers workspaces

| Raccourci | Action |
|-----------|--------|
| `Super` + `Shift` + `1` à `0` | Déplacer la fenêtre vers le workspace 1 à 10 |

## Session et système

### Contrôle de session

| Raccourci | Action |
|-----------|--------|
| `Super` + `L` | Verrouiller l'écran |
| `Super` + `Shift` + `L` | Quitter Hyprland |
| `Super` + `Ctrl` + `Shift` + `R` | Redémarrer le système |
| `Super` + `Ctrl` + `Shift` + `O` | Éteindre le système |

### Configuration et scripts

| Raccourci | Action |
|-----------|--------|
| `Super` + `Ctrl` + `R` | Recharger Hyprland, Waybar et SwayNC |
| `Super` + `N` | Afficher/Masquer le centre de notifications |
| `Super` + `H` | Afficher l'aide des raccourcis |
| `Super` + `Shift` + `S` | Recharger le thème actuel |
| `Super` + `Ctrl` + `S` | Sélecteur de thèmes |

## Captures et enregistrements

### Captures d'écran

| Raccourci | Action |
|-----------|--------|
| `Super` + `T` | Capture de zone vers presse-papiers |
| `Super` + `S` | Flameshot (capture annotée) |
| `Super` + `Shift` + `T` | Capture vers notes |

### Enregistrements vidéo

| Raccourci | Action |
|-----------|--------|
| `Super` + `F11` | Enregistrer l'écran complet avec audio |
| `Super` + `F12` | Enregistrer une zone avec audio |

## Zoom du curseur

### Contrôle du zoom

| Raccourci | Action |
|-----------|--------|
| `Super` + `+` | Augmenter le zoom de 10% |
| `Super` + `-` | Diminuer le zoom de 10% |
| `Super` + `Ctrl` + `Alt` + `Molette` | Zoomer/Dézoomer avec la molette |
| `Ctrl` + `Molette` | Zoomer/Dézoomer avec la molette |

### Réinitialisation du zoom

| Raccourci | Action |
|-----------|--------|
| `Super` + `Shift` + `-` | Réinitialiser le zoom à 100% |
| `Super` + `Shift` + `0` | Réinitialiser le zoom à 100% |
| `Super` + `Shift` + `Molette` | Réinitialiser le zoom à 100% |
